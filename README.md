# 복슬금융계산기

## 1. 주요기능

경제 금융활동에 필요한 각종 계산기 제공. 계산에 필요한 공식을 제공해 검증을 할 수 있음.

## 2. 실행 및 빌드

### 2.1. 초기셋팅

```shell
$ npm install
```

### 2.2. 실행 방법

```shell
$ npm run dev
```

### 2.3. 빌드 방법

```shell
$ npm run build
```

- 빌드 결과는 `/.next` 폴더에 생성됨
